🏈 GetDraft – Complete Web App Overview

1. General Presentation

GetDraft is a swipe-based sports networking platform designed to connect:

Athletes

Coaches

Agents

Parents

Its main goal is to help sports talents find real opportunities through a modern matchmaking system inspired by social dating apps.

The platform promotes itself with the tagline:

“Where Talent Meets Opportunity”

It presents itself as a global ecosystem where athletes can showcase their skills and get discovered.

2. Main Concept of the Platform

GetDraft is built around three core actions:

Swipe → Match → GetDraft

Swipe
Users browse profiles of other members (players, coaches, agents) and can like or pass.

Match
When two users like each other, a connection is created.

GetDraft
Matches can turn into real opportunities such as recruitment, scholarships, or professional contacts.

This mechanism is the foundation of the whole application.

3. Supported Sports Levels

The platform targets athletes from multiple competitive levels:

Little League

High School

NCAA

USports

Professional

Olympics

This shows that the app is designed for both young talents and high-level professionals.

4. Global Network Section

GetDraft highlights strong community statistics:

10,000+ active users

50+ countries

1,000,000+ matches made

This is used as social proof to show the platform’s credibility and reach.

📝 User Workflow 5. Sign Up Process

The registration workflow is structured in multiple steps:

Step 1 – Create Account

User must provide:

Email address

Password

Password confirmation

Choose a role

Available roles:

Player / Athlete

Coach

Agent

Parent

Step 2 – Email Verification

A 6-digit code is sent to the user email to confirm identity.

Only after validation can the user continue.

6. Subscription Plans

After signing up, the user is asked to choose a plan.

Available Plans
Plan Price / Month Swipes per day Features
Basic $0 10 Text messaging
Starter $3 30 Enhanced messaging
Pro $7 100 Advanced features
Premium $15 Unlimited Full access

The Pro plan is highlighted as the most popular.

Users can start with the free Basic plan and upgrade later.

7. Profile Completion

Once the plan is chosen, the user must complete their profile.

Required Information

For the “Player/Athlete” role:

Personal details

Sport information

Physical attributes

Performance statistics

Position

Experience level

Highlight Reel

User can add a video link from:

YouTube

Vimeo

Hudl

MaxPreps

This allows athletes to showcase their skills.

Identity Verification

There is a section requiring:

Upload of an official ID (Passport / Driver’s License)

This is used to prevent fake profiles and ensure platform safety.

A progress bar shows:

“Profile X% Complete – Complete 60% more to start swiping”

Once enough information is added, the user can proceed.

9. Main Navigation Areas

Once inside the platform, several main pages are available.

9.1 Discover (Swipe Interface)

This is the core page:

Cards of profiles appear

Each card shows:

Name

Location

Role

User actions:

❤️ Like (Swipe Right)

❌ Pass (Swipe Left)

9.2 Dashboard

The dashboard provides an overview:

Total matches

Active chats

Profile views

Weekly views

Profile strength percentage

Quick actions:

Start Swiping

This acts as the user control center.

9.3 Matches

This page lists all users with whom a mutual match occurred.

“No matches yet – Start swiping!”

9.4 Messages

Messaging system where matched users can chat.

“No matches yet to message.”
